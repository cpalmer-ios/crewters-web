Crewters provides a robust iOS mobile app and a RESTful API, with API documentation available via [Swagger](https://swagger.io/) at http://localhost:9000/api/docs.

## App Features

The Crewters iOS Mobile App has the following features out of the box:

1. Create & Join Sport Events
<span className="text-sm text-muted-foreground">• Users can create new sport events, join existing ones, and manage their participation.</span>
2. User Profiles
<span className="text-sm text-muted-foreground">• View and edit user profiles, including bio, stats, sports history, and show reels (YouTube/Vimeo).</span>
3. Friend System
<span className="text-sm text-muted-foreground">• Add friends, view friend profiles, and see their event history and achievements.</span>
4. Game History & Progress
<span className="text-sm text-muted-foreground">• Track your event participation and progress for each sport with visual progress lines.</span>
5. Notifications
<span className="text-sm text-muted-foreground">• Receive and view notifications for friend requests, event invites, and more.</span>
6. Interactive Map & Listings
<span className="text-sm text-muted-foreground">• Browse and interact with events on a map and in a carousel listing.</span>
7. Custom Icons & Theming
<span className="text-sm text-muted-foreground">• Sports are visually represented with custom icons and dynamic theming.</span>
8. Teams
<span className="text-sm text-muted-foreground">• Join a team or create your own.</span>
9. Leagues
<span className="text-sm text-muted-foreground">• Compete in a league or create your own.</span>
9. Trophies
<span className="text-sm text-muted-foreground">• Earn trophies as you progress with your fitness goals.</span>
10. Search
<span className="text-sm text-muted-foreground">• Built-in search for events and users based on geolocation (in progress).</span>
11. Dark Mode
<span className="text-sm text-muted-foreground">• Dark mode support for comfortable viewing (in progress).</span>
